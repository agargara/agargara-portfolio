  • Agargara Music

    A personal music website built with Wordpress and custom templates. Styled with a modified version of Daryl Koopersmith's Hum theme.

  • Piston Source Records

    A netlabel dedicated to releasing music made with Piston Collage. Written in 2008, the techniques used on this website are now somewhat dated, but the site is completely hand-coded with valid HTML and CSS. Styled to look like the program Piston Collage.

  • Film Score Generator

    A Max/MSP/Jitter program that automatically scores videos. A database of music clips is analyzed and tagged with various aspects. The program analyzes a video as it plays and tries to select appropriate music clips from the database to match. Watch a demo.

  • Dodge City

    A small game about dodging falling cubes. The graphics engine is coded from scratch. Download and play (executable .jar)

  • Musical L-Systems

    Lindenmayer systems, or L-systems, are grammars that are usually used to model organic structures. This is a Max/MSP program that uses the production rules of L-Systems to generate musical sequences. Download the source.
